# Auth Components Package

Chứa các UI Component nội bộ chỉ phục vụ riêng cho module Xác thực (Auth).

## Ví dụ
- `LoginForm.tsx`, `RegisterForm.tsx`, `ForgotPasswordForm.tsx`.
- Các component này chỉ phục vụ riêng mục đích xác thực và không được sử dụng ở các module nghiệp vụ khác.
- Đặt tên file theo chuẩn `PascalCase`.
