# Constants Directory

Lưu trữ các hằng số dùng chung cho toàn bộ dự án.

## Quy tắc thiết kế
- Chứa các định nghĩa biến cấu hình như `API_BASE_URL`, `TOKEN_KEY`, định nghĩa các cấu hình phân trang, cấu hình định dạng, v.v.
- Đặt tên biến hằng số theo kiểu chữ hoa ngăn cách bởi dấu gạch dưới (`SCREAMING_SNAKE_CASE`).
