# Configuration Directory

Chứa các tệp cấu hình toàn hệ thống.

## Nội dung
- Cấu hình client (ví dụ: axios interceptors, thiết lập các thư viện ngoài như Ant Design theme, firebase...).
- Tránh viết logic nghiệp vụ hoặc UI component trong thư mục này.
- Đặt tên file sử dụng chữ thường đầu từ (`camelCase`).
