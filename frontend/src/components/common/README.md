# Common Components Package

Chứa các thành phần UI nhỏ, độc lập, có thể tái sử dụng ở nhiều vị trí trong dự án.

## Ví dụ
- `Button.tsx`, `Input.tsx`, `Checkbox.tsx`, `Table.tsx`, `Badge.tsx`, `Modal.tsx`.
- Các component này chỉ nhận props và render giao diện, không chứa logic nghiệp vụ liên kết với API hay database.
- Đặt tên file theo chuẩn `PascalCase`.
