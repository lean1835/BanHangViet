# Layouts Package

Chứa các khung layout chính bao bọc các trang hiển thị của ứng dụng.

## Ví dụ
- `MainLayout.tsx`: Chứa khung sidebar, header và vùng nội dung chính cho các trang quản trị.
- `AuthLayout.tsx`: Khung giao diện cho các màn hình đăng nhập, đăng ký, quên mật khẩu.
- Đặt tên file theo chuẩn `PascalCase`.
