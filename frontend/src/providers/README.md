# Providers Directory

Chứa các Provider bọc ngoài ứng dụng để cung cấp context hoặc cấu hình dùng chung.

## Ví dụ
- `ThemeProvider.tsx`: Quản lý cấu hình giao diện (Sáng/Tối, Ant Design Tokens).
- `AuthProvider.tsx`: Quản lý thông tin đăng nhập toàn cục.
- `QueryProvider.tsx` / `ReduxProvider`: Cấu hình store.
- Đặt tên file theo chuẩn `PascalCase`.
